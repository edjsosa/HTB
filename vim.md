# Vim

open file with vim
`vim file`

enter insert mode
`esc+i`

back to normal mode
`esc`

Cut character
`x`

Cut word
`dw`

Cut full line
`dd`

Copy word
`yw`

Copy full line
`yy`

Paste
`p`

Go to line number 1.
`:1`

Write the file 'i.e. save'
`:w`

Quit
`:q`

Quit without saving
`:q!`

Write and quit
`:wq`
